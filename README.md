# My-diary-printer
هو موقع سويته بحيث يرتب ويكون تطوير لمذكراتي اليومية الي اكتبها كل يوم بحيث منه بدل ما اكتبها يدوياً اقدر اكتبها في الصفحه بالخط الي انا ابي من بين خطوط و وواقدر اطبعها بكل سهوله a4 او a6 واقدر وحطيت نضام النقاط الي كنت احطه يدوياً ولي هو مجرد مهام يوميه اقيم فيها نفسي وايضاً امكانيه وضع التاريخ في كل صفحة وحليت المشكله الجمل الطويله التي كانت لا تكفي الصفحه كامله بحيث صار هنا إذا كتبت كلام كثير يصغر الخط في كل مذكرة

This is a website I created to organize and improve my daily journal entries. Instead of writing them by hand, I can now write them on the page in my preferred font, and easily print them in A4 or A6 format. I've also added the point system I used to manually create, which is simply a daily task for self-assessment. Additionally, I can add the date to each page, and I solved the problem of long sentences not fitting on the entire page; now, if I write a lot, the font size decreases in each entry.
