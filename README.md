# My-diary-printer
هو موقع سويته بحيث يرتب ويكون تطوير لمذكراتي اليومية الي اكتبها كل يوم بحيث منه بدل ما اكتبها يدوياً اقدر اكتبها في الصفحه بالخط الي انا ابي من بين خطوط و وواقدر اطبعها بكل سهوله a4 او a6 واقدر وحطيت نضام النقاط الي كنت احطه يدوياً ولي هو مجرد مهام يوميه اقيم فيها نفسي وايضاً امكانيه وضع التاريخ في كل صفحة  واضا كل
